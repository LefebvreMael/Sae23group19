Accueil : Banque de Films

- Bienvenue sur la page d'accueil de notre filmographie. Cette page été créée dans le cadre de la Saé 2.03 "Mettre en place une solution informatique pour l'entreprise". Le sujet qui nous a été attribué est la filmographie. Il nous est demandé de fournir un interface en web dynamique. Celui-ci doit comprendre de multiples critères, c'est pour cela que je vais vous présenter les fonctionnalités de la page.

- Sur la partie gauche de votre écran vous trouverez un bandeau de navigation entre les différentes pages, chaque bouton mène a une page différente. Un bouton par thème et dans chacun il y aura un moyen d'ajouter un objet à la liste.

- Une fois sur la page qui liste les films, vous avez la possibilité de cliquer sur le nom du film pour pouvoir afficher les détails le concernant.


Merci de nous avoir accordé votre attention, le groupe 19 vous souhaite un très bon visionnage.
